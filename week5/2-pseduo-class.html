Pseudo class

Pseudo class allow for adding of state to an HTML elmemet.

The most common place where we add Pseduo class is to a link.

Link default state

Links have three states by default

  1. Normal/unvisited (blue) - if the value in the href is not in the local browser history.
  2. visited (purple) - if the value in the href is in the local browser history.
  3. active (red) - when the user is activly press the link

When you style a link and do not define the pseudo classes they will all display with the styles that you define on the link.
